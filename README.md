## Objetivo do Projeto

Este repositório contém uma **simulação didática** de:

- Um **Ransomware Simulado**: opera apenas em um **arquivo de teste** dentro da pasta do projeto, aplicando uma transformação simples no conteúdo e exibindo uma mensagem de “resgate”;
- Um **Pseudo-Keylogger**: registra **entradas de texto voluntárias** do usuário (via `input()`) em um arquivo `.txt`, simulando o conceito de captura de dados, sem capturar teclas do sistema nem rodar de forma oculta.

O foco principal é:

- Compreender o funcionamento **conceitual** de ransomware e keyloggers;
- Praticar programação em Python em **ambiente seguro**;
- Refletir sobre estratégias de **defesa e prevenção**;
- Utilizar o GitHub como **portfólio técnico**, com documentação clara.
